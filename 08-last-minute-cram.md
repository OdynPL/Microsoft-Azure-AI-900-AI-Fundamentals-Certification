# 8. Szybka ściąga i pułapki egzaminacyjne AI-900

## Najważniejsze różnice i pułapki
- Computer Vision ≠ NLP ≠ Generatywna AI – rozpoznawaj typ workloadu po opisie zadania
- OCR = rozpoznawanie tekstu na obrazach, nie analiza sentymentu
- Sentiment analysis = NLP, nie Computer Vision
- GPT = generatywna AI, nie klasyczne ML
- Responsible AI = etyka, bezpieczeństwo, przejrzystość

## Szybkie porównania usług
| Zadanie | Usługa Azure |
|---------|--------------|
| Klasyfikacja obrazów | AI Vision |
| Detekcja twarzy | AI Face |
| Analiza tekstu | AI Language |
| Rozpoznawanie mowy | AI Speech |
| Generowanie tekstu | OpenAI |
| Katalog modeli | Foundry |

## Strategie egzaminacyjne
- Czytaj uważnie scenariusz – kluczowe są słowa typu: "klasyfikacja", "generowanie", "ekstrakcja"
- Eliminuj odpowiedzi, które nie pasują do typu workloadu
- Responsible AI – zawsze wybieraj odpowiedzi promujące etykę i bezpieczeństwo
